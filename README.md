# terminal-portfolio

A minimal, ultra-modern terminal-themed personal portfolio. Built with pure HTML, CSS, JavaScript, and George's signature CSS glitch effect. No frameworks, no bloat, pure spite.

## Features
- **Visual Aberration Glitch Header**: Adaptable, desynced horizontal slice keyframe glitch using pure CSS.
- **Interactive Console Widget**: Works as a fully functioning terminal emulator supporting commands like `about`, `skills`, `writings`, `discord`, `socials`, `motd`, and `clear`.
- **Status Badges**: Custom pure CSS status badges representing tech stacks, systems, and developer philosophy.

## How to Run & Preview
Since this site is static with zero dependencies, you can preview it immediately:
1. Open `index.html` directly in any web browser.
2. Alternatively, run a local development server:
   ```bash
   python3 -m http.server 8000
   ```
   and navigate to `http://localhost:8000`.
